## Pines

Pines是一个基于 Go+Vue 构建的对象存储服务管理应用，目前集成了阿里云OSS，腾讯云COS，又拍云云存储。

### 构建

1. 重命名 config.yaml.example 为 config.yaml

2. 修改 config.yaml 内容

3. 从源代码运行或下载release版本直接运行

```bash
# 从源代码运行
go run .
```
